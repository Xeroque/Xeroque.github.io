   The logo is my favourite part of this project. It follows the brand name Mel Sol, which translates to Honey Sun to create a animated version of it that also resembles a freshly backed sour dough bread still radiating heat.This packaging concept was created and rendered using Adobe Dimension.



   The website was coded from scratch in HTML,CSS,JS and Ruby and hosted on Github Pages using Vercel deployment. All technologies I'm familiar with.
